# archive-home-portal

## 项目简介

`archive-home-portal` 是一个用于归档和展示多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为一份静态 HTML 资源的集合，方便浏览、索引与版本管理。

## 目录说明

```
archive-home-portal/
├── pages/          # 存放各个独立 HTML 页面文件
├── assets/         # 共用资源，如样式、脚本、图片等
├── index.html      # 入口页面，用于索引和导航
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件代表一个独立的归档页面。
- `assets/`：存放多个页面可能共用的 CSS、JavaScript、字体或图片文件。
- `index.html`：仓库的主入口，提供到各个归档页面的链接或摘要。

## 页面归档说明

- 所有归档页面均为自包含的 HTML 文件，可在本地直接打开浏览。
- 页面内容可能来自不同来源或时期，仅作存档与展示用途。
- 部分页面可能引用外部资源（如字体库、公共 CDN），请确保网络畅通。

## 维护说明

- 新增页面：在 `pages/` 下创建新目录或文件，并更新 `index.html` 中的索引。
- 更新页面：直接替换对应 HTML 文件，建议保留历史版本记录（通过 Git 管理）。
- 资源管理：共用资源请放入 `assets/`，避免在多个页面中重复存放。

## 开发与贡献

欢迎通过 Issue 或 Pull Request 提出改进建议。提交新页面时请尽量保持文件结构清晰，并确保 HTML 内容符合基本语法规范。
